Exercise 1 (Github Account)

Create a Github account and provide us with the user name:

  1. Create a free account at Github, if you do not have one already.

  2. Once created, go to this link and complete the form before Tuesday August 30th 11pm.

  3. Once you successfully complete step 2, look out for an invitation email from Github in the mailbox of the email address you used to sign up for Github. Accept the invitation to join the course Github organization. Also, a private repository should be created for you that you will use as your personal space for the exercises and homeworks throughout the semester.
